# FrankX Claude Code Ecosystem - Landing Page

This directory contains the GitHub Pages landing page for the FrankX Claude Code ecosystem.

## Viewing the Site

Once published, the site will be available at: https://frankxai.github.io/claude-complete

## Local Development

To view locally, simply open `index.html` in your browser.

## Publishing

GitHub Pages will automatically serve the `index.html` file from this `/docs` directory once you:

1. Go to repository Settings â†’ Pages
2. Select Source: "Deploy from a branch"
3. Select Branch: "main" and folder: "/docs"
4. Save

The site will be live within a few minutes.

## SEO Optimization

The landing page includes:
- Semantic HTML structure
- Meta tags for search engines
- Open Graph tags for social sharing
- Twitter Card tags
- Mobile-responsive design
- Fast load times (no external dependencies)

## Updates

To update the landing page, edit `index.html` and commit to the main branch. Changes will be live within minutes.
