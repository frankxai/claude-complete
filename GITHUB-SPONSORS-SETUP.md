## Step 1: Enable GitHub Sponsors

1. Go to https://github.com/sponsors
2. Click "Join the waitlist" (if not enabled) or "Set up GitHub Sponsors"
3. Complete your profile:
   - Select account: **frankxai** (or your personal account)
   - Complete fiscal host information
   - Link Stripe/PayPal for payments
